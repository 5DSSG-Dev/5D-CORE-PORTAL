# 5D Core Portal

5D Service Solutions Global Core Portal is the first public website foundation for the 5D ecosystem.

The site is built as a luminous exploratory portal rather than a traditional brochure. It presents the company identity, realm architecture, AEGIS doctrine, CyberPeer pathway, DataQuad concept, and Omni OS bridge through an interactive interface.

## Operating Principle

Human Centered, AI Respected.

## Public Posture

Presented for your Consideration.

## Tech Stack

- React
- Vite
- Framer Motion
- Lucide React
- CSS custom properties

## Run Locally

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
npm run preview
```

## Current Phase

Core Portal v0.1 establishes the first website foundation. It will evolve into official assets, route-level pages, DataQuad viewer, CyberPeer profiles, and teleport transitions.
